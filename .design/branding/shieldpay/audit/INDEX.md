# ShieldPay Brand Audit

> Phase: audit | Brand: ShieldPay | Generated: 2026-06-27

---

Audit of the existing ShieldPay brand identity for evolution, not rebrand.
ShieldPay is confidential payroll for DAOs/Web3 teams on Stellar + zero-knowledge
proofs, across three portals (company/CFO, worker, auditor). The governing
principle is "cryptography is invisible": finance-grade and calm, never
neon/hacker. The brief's complaint, confirmed here: the UI works but feels bland
and lifeless, one craft tier below the market bar it set (Stripe, Mercury, Deel,
Ramp, Coinbase, Linear, QuickBooks).

## Chunks

1. [brand-inventory.md](./brand-inventory.md) — full catalog: mark, color tokens

5. [evolution-map.md](./evolution-map.md) — element-by-element PRESERVE / EVOLVE /
   REPLACE with persona-tied rationale and percentages.

## Headline findings

**Top strengths**
- A distinctive, story-bearing indigo-to-emerald gradient ("protected, then
  proven") — the one truly ownable asset.
- Correct, on-trend calm dark slate palette that earns CFO trust against neon
  crypto tools.
- Clean, enforced token architecture (CSS vars + semantic Tailwind) — the safe
  surface to evolve on.
- A clear, well-held strategic principle: "cryptography is invisible."

**Top weaknesses**
- Flat: no elevation system; three slate surfaces separated by a hairline, so the
  UI reads undifferentiated rather than calm.
- One-rung text hierarchy (98% then 62%) — dense payroll/audit data lacks the
  legibility tiers of Ramp/Stripe.
- Inert: a single fade and brightness hovers; the trust-critical payroll progress
  sequence has no motion.
- The gradient is used as flat paint, and figures are not yet mono/tabular, so the
  "exact, verifiable" promise is not delivered at the pixel level.

**Headline calls** — PRESERVE ~38% / EVOLVE ~44% / REPLACE ~19%
- PRESERVE: the gradient motif, slate hues, the verified badge, Inter, the
  invisible-crypto rule, the token architecture.
- EVOLVE: gradient-as-light, the shield glyph (negative-space privacy slit), text
  tiers, shadow deployment, mono/tabular figures, motion + payroll stepper.
- REPLACE: the flat elevation system, the dead `accent` token, and the stale
  legacy tagline / pipe-separator copy.

A clean evolution: protect the identity and architecture, build up the missing
craft layer, replace only inert scaffolding. No rebrand needed.
