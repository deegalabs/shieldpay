# Evolution Map

> Phase: audit | Brand: ShieldPay | Generated: 2026-06-27

---

Element-by-element disposition. Each rationale ties to a persona: CFO/Ops
(skeptical, wants premium trust), Worker (wants a clear, trustworthy record),
Auditor (wants exact, legitimate, precise figures).

## Map

| Element | Current state | Call | Rationale (persona) |
| --- | --- | --- | --- |
| Indigo to emerald gradient | Flat text fill + faint corner glows | **PRESERVE** | The only un-generic equity. Indigo reassures the privacy-anxious CFO, emerald confirms settlement for worker/auditor. Keep the motif; deepen its use (see below). |
| Gradient *application* | Decorative paint | **EVOLVE** | Re-deploy as dimensional light (surface edges, focused fields, hero figure) so the CFO reads "premium," matching Mercury/Coinbase accent craft. |
| Shield-check mark | Near-stock lucide glyph | **EVOLVE** | Keep the shield silhouette + gradient (recognition) but cut the check as a negative-space privacy slit so it stops looking generic to a CFO comparing tools. |
| Calm dark slate palette (hues) | `#0A0F1E` to `#18202F`, correct | **PRESERVE** | The anti-neon calm is exactly what lets a skeptical CFO trust a crypto-adjacent product. Do not change the feel. |
| Surface / elevation system | 3 flat steps, border-only depth | **REPLACE** | Biggest craft liability. Add surface-light layering + graded shadow so dense payroll/audit tables gain hierarchy (CFO, Auditor) instead of reading flat. |
| Text hierarchy tokens | 2 rungs (98% then 62%) | **EVOLVE** | Add a mid `foreground-subtle` tier so dense tables feel calm and scannable for the Auditor and CFO, matching Ramp/Stripe. |
| `accent` token | Dead duplicate of `brand` | **REPLACE** | Repurpose as a real secondary/elevation or hover token; no equity lost. |
| Shadow vocabulary (`elevated`, `glow`) | Defined, unused | **EVOLVE** | Actually deploy graded shadows so elevation is visible, giving the CFO the "premium tool" depth cue. |
| Inter typeface | Single family, no equity | **PRESERVE** | Correct, free, fintech-legible. Keep; optionally pair a display/mono face later. No persona needs a change here. |
| Tabular / mono on figures | Intended, not applied | **EVOLVE** | Apply mono + tabular to every amount, hash, proof ID. Makes figures *feel* exact for the Auditor and trustworthy for the Worker, delivering the strategy's own promise. |
| Verified emerald badge | Good, slightly plain | **PRESERVE** | The recurring proof-of-trust moment across all three portals. Keep meaning; polish finish only. |
| Motion system | Single `fade-in`, inert buttons | **EVOLVE** | Add confirm-motion, especially an animated payroll stepper (sending to proving to verifying to settled). This is the CFO's trust-critical moment; today it is silent. Gate on `prefers-reduced-motion`. |
| Ambient body glows | Faint corner radials | **EVOLVE** | Keep the idea, strengthen and integrate with the new elevation light so it reads as intentional atmosphere, not noise. |
| "Cryptography is invisible" rule | Enforced, working | **PRESERVE** | Core strategic equity; separates ShieldPay from every hacker-aesthetic crypto tool for all three personas. |
| Landing metadata tagline | Legacy legal-proof line | **REPLACE** | Stale positioning; replace with the confidential-payroll line so the CFO gets one consistent story from tab to hero. |
| Landing prose `|` separators | Literal pipes inside sentences | **REPLACE** | Polish miss on the most-seen surface; replace with proper punctuation. |
| Token architecture (CSS vars + semantic Tailwind) | Clean, enforced | **PRESERVE** | The system's real strength and the safe surface to evolve on. Do not break it. |

## Summary disposition

Counting the 16 elements above:

- **PRESERVE: 6** (~38%) — gradient motif, slate hues, Inter, verified badge,
  invisible-crypto rule, token architecture.
- **EVOLVE: 7** (~44%) — gradient application, shield glyph, text hierarchy,
  shadow deployment, mono/tabular figures, motion, ambient glows.
- **REPLACE: 3** (~19%) — flat elevation system, dead `accent` token + legacy
  tagline + pipe separators (grouped as plumbing/copy replacements).

The shape is a true evolution: keep the brand's identity and architecture, build
up the missing craft layer (depth, hierarchy, precision, motion), and replace
only inert scaffolding. No rebrand, no equity loss.

## Sequencing hint (highest leverage first)

1. Elevation/surface-light system + shadow deployment (fixes the headline "flat").
2. Text-tier + tabular/mono figures (fixes data legibility and "exact" promise).
3. Gradient-as-light re-application + shield glyph evolution (lifts premium feel).
4. Confirm-motion + payroll stepper (lands the trust moment).
5. Copy cleanups (tagline, pipes) — cheap, do alongside.
